# 📖 Sobre o Projeto

O **Sistema SalgAki** é um projeto acadêmico desenvolvido com o objetivo de aplicar, na prática, os conhecimentos adquiridos durante a graduação em **Análise e Desenvolvimento de Sistemas**.

A aplicação simula o gerenciamento de uma salgaderia, oferecendo funcionalidades para cadastro de clientes, produtos e pedidos, permitindo organizar o fluxo de vendas de forma simples e eficiente.

Durante o desenvolvimento foram aplicados conceitos de:

- Programação Orientada a Objetos (POO);
- Desenvolvimento Web;
- Arquitetura MVC;
- Persistência de Dados;
- Banco de Dados Relacional;
- Boas práticas de programação.

---

# 🎯 Objetivos

- Desenvolver uma aplicação completa utilizando tecnologias atuais;

# 🚀 Como Executar

## 1. Clonar o repositório

```bash
git clone https://github.com/Emanuela-Azevedo/Sistema-SalgAki.git
```

## 2. Acessar a pasta

```bash
cd Sistema-SalgAki
```

## 3. Configurar o banco de dados

- Criar o banco MySQL;
- Configurar as credenciais no arquivo de propriedades da aplicação;
- Executar os scripts SQL (caso existam).

## 4. Executar o projeto

Abra o projeto em uma IDE compatível (IntelliJ IDEA, Eclipse ou VS Code) e execute a aplicação.
